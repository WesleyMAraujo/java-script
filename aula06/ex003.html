Tipos primitivos

em caso de apresentação de soma, se feito de forma normal os numeros irão apenas concatenas.

podemos resolver com um cast

var n1 = window.prompt('digite um numero')

var n1 = Number.parseFloat(window.prompt('digite um numero'))

ou podemos simplesmente usar Number(window.prompt('digite um numero'))

para converter para string basta apenas usar string(s)

para imprimir resultados com varias variaveis usando CRASE: ´´. `O aluno ${nome} com ${idade} anos tirou a nota ${nota}`

var n1 = Number(window.prompt('digite um numero'))
var n2 = Number(window.prompt('digite outro numero'))
var s = n1 + n2
window.alert(`A soma entre ${n1} e ${n2} é igual a : ${s}`)

s.toUpperCase() tupara para maiusculas
s.toLowerCase() tudo para minuscula
